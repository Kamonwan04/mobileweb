รายละเอียดงาน

การพัฒนาแอปด้วย Ionic ร่วมกับ Firebase Firestore

  • สร้าง Ionic Application และเชื่อมต่อ Firebase
  • ใช้งาน Firestore Database สำหรับจัดเก็บข้อมูล
  • พัฒนาแอปที่รองรับการเพิ่ม แสดง แก้ไข และลบข้อมูล (CRUD)
  • เข้าใจการทำงานของ Realtime Database

1. เพิ่มข้อมูล (Create)

2. แสดงข้อมูลแบบ Realtime (Read)

3. แก้ไขข้อมูล (Update)

4. ลบข้อมูล (Delete)
