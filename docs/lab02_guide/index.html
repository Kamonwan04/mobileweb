Highlights of Mor Din Daeng

ศูนย์อาหารและบริการ (KKU Complex)

"คอมเพล็กซ์" เป็นศูนย์กลางไลฟ์สไตล์ของชาว มข. ครบครันด้วยร้านอาหารอร่อย ธนาคาร ร้านหนังสือ และพื้นที่กิจกรรม บรรยากาศคึกคักตลอดวัน เป็นจุดนัดพบสำคัญที่ไม่ควรพลาด

ศูนย์ประชุมอเนกประสงค์กาญจนาภิเษก

แลนด์มาร์คริมบึงสีฐาน โดดเด่นด้วยสถาปัตยกรรมทรงไทยประยุกต์ที่งดงาม เป็นสถานที่รับพระราชทานปริญญาบัตร และจุดชมวิวพระอาทิตย์ตกที่สวยที่สุดแห่งหนึ่งในมหาวิทยาลัย

ศาลเจ้าพ่อมอดินแดง

สิ่งศักดิ์สิทธิ์คู่ใจชาว มข. ตั้งอยู่ริมถนนมิตรภาพ เป็นที่ยึดเหนี่ยวจิตใจของนักศึกษาและบุคลากร ผู้คนนิยมมากราบไหว้ขอพรเพื่อความเป็นสิริมงคลในทุกโอกาสสำคัญ

อุทยานเทคโนโลยีการเกษตร (สวนเกษตร)

ปอดขนาดใหญ่ของมหาวิทยาลัย เต็มไปด้วยทุ่งดอกไม้ตามฤดูกาลและแปลงเกษตรสาธิต เหมาะสำหรับการมาออกกำลังกาย ถ่ายรูป หรือพักผ่อนหย่อนใจท่ามกลางธรรมชาติ
